In order to run the code on Google Colaboratory, please uncomment the package download line and follow these steps:
1) Go to the Colab website (https://colab.research.google.com/) and sign in with a Google account.
2) Upload the notebook by clicking on the "Upload" button.
3) Start running cells by selecting a cell and clicking the "play" button or pressing Shift + Enter.
4) Connect to a runtime by clicking "Runtime" > "Run all".
